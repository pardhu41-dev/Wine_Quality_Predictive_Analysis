# Wine Data Analysis — Predictive Insights 2024

This is an end-to-end data science project utilizing Python to analyze wine quality datasets, uncover key physicochemical features influencing quality scores, and predict wine categories using machine learning classifiers.

## Key Features
* Exploratory Data Analysis: Handled data exploration and visualized feature correlations using Seaborn heatmaps to understand chemical relationships.
* Feature Engineering: Binarized the wine quality ratings into distinct classes (Good vs. Bad) and normalized the feature sets using StandardScaler.
* Predictive Modeling: Built and evaluated Logistic Regression and Random Forest classifiers, optimizing hyperparameters to achieve strong predictive accuracy.

## Tech Stack
* Language: Python
* Libraries: pandas, NumPy, scikit-learn, Matplotlib, Seaborn
* Environment: Jupyter Notebook

## Results and Performance Evaluation
* Logistic Regression Accuracy: ~75%
* Random Forest Accuracy: ~80% (Best Performing Model)
